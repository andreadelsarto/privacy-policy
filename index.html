Italiano

Task è un’app di gestione attività che funziona interamente in locale sul dispositivo.

  • Nessuna raccolta dati: l’app non raccoglie, condivide o invia dati personali a server esterni.

  • Permessi: l’app non richiede permessi sensibili. Eventuali funzioni opzionali (es. notifiche) operano localmente.
  • Sicurezza: i dati locali sono accessibili solo sul dispositivo. Consigliamo di proteggere il dispositivo con metodi di sicurezza nativi.

English

Task is a task management app that works entirely offline on the device.

  • No data collection: the app does not collect, share, or transmit personal data to external servers.
  • Local storage: all user data stays on the device and can be deleted from within the app or by uninstalling it.
  • No tracking/third-party SDKs: we do not use analytics, advertising, or external services.
  • Permissions: the app requires no sensitive permissions. Any optional features (e.g., notifications) operate locally.
  • Security: local data is accessible only on the device. We recommend using the device’s native security protections.
